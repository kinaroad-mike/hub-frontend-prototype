# hub-frontend-prototype

## Setup Development Environment
### 1. Clone Repository
```bash
git clone git@github.com:kinaroad-mike/hub-frontend-prototype.git
```
### 2. Install Packages
- Install [Node.js](https://nodejs.org/en/)
- Install project dependencies
```bash
$ cd <rootdir>
$ npm install
```

## Start Prototype App
```bash
$ npm start
```
This will run the webpack server allowing access to the app by
 visiting [http://localhost:8080/login](http://localhost:8080/login)
## Test Prototype App
```bash
$ npm test
$ npm test -- --coverage
```
